Cheats and Commands

To open the console press CTRL + SHIFT + C. From there type testingcheats {on/off}. One complete you will have access to all the cheats.

Money Cheats
CommandDescription
kaching Get 1,000 Simoleons
rosebud Get 1,000 Simoleons
motherlode Get 50,000 Simoleons
Money {number} Set funds to a specific number
FreeRealEstate {on/off} Make all the lots free when in world map view
Build Mode Cheats
CommandDescription
bb.moveobjects Move objects without following place rules
bb.ignoregameplayunlocksentitlement Unlock all career items
SHIFT + ] Increase item size
SHIFT + [ Decrease item size
Click roof and press SHIFT + C More options for moving a roof
bb.showhiddenobjects Unlock all debug items
bb.showliveeditobjects Unlocks extra world environment items. For this command you will need bb.showhiddenobjects on
bb.enablefreebuild Be able to edit to edit any lot other than your own
